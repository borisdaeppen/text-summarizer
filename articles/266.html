Error Message: “...” is not a valid Link/Rotator ID in account “$username”

This error message means that you’ve used a Link or Rotator ID that does not exist in the account.

If you recall, when you create a tracking link (or rotator link), you need to specify a Link ID that you must use on the end of the link for it to work correctly:
 


 
For the link to work correctly, you must use the Link ID or Rotator ID exactly as you entered when you created the link, including how you capitalized each letter.

If a click is delivered to your account and the Link ID or Rotator ID does not exactly match any link in your account, you will get the “not a valid Link/Rotator ID” message.


Common Reasons for this Error

You can get this error for a variety of reasons:
 
1.    There’s a typing mistake in your Link ID or Rotator ID ...
 
2.    You’ve deleted the link so it literally does not exist anymore ...
 
3.    You’ve recently changed the username for your account, but you forgot to update your links to use the new username. Remember that your generic links contain your username:

http://www.clkmg.com/$username/linkID

Also, if you’ve changed your username and you’re using Script Forwarding, you’ll need to regenerate the two control files you created before. You can do that here:
