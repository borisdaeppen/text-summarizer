How can I add Facebook pixels to tracking links?

Very easily!

You can add Facebook pixels to your tracking links the same exact way you would add ClickMagick tracking pixels to a tracking link.

Please see the following video that shows how to do that in detail:
 
Add a Tracking Pixel Using a Tracking Link

Basically, when you click on the “Advanced Settings” of any tracking link, you should see a “Pixel/Code” box where you can add your code to.

If you add your Facebook pixel into that area, you should be able to track conversions on your Facebook end and even create a conversion objective campaign!
