How do I set up a Postback URL with ClickBank?

ClickBank doesn’t use the term “Postback URL” so you won’t find that mentioned in any of their videos or documentation.

Instead, they call this URL the “Instant Notification URL”.

The industry-standard term for this concept is “Postback URL” so that’s what ClickMagick uses in its videos and documentation, but you can think of the terms interchangeably once they are set up.

One difference in the set up process between standard Postback URLs and ClickBank’s Instant Notification URL is that ClickBank introduced a “Secret Key” for added security.

Creating the “Secret Key” is just a simple one-time extra step you need to take when setting up the “Postback URL” for ClickBank and we have a separate FAQ that will walk you through the entire process:
 
  How to I set up ClickBank’s Instant Notification URL?

Again, just keep in mind that if you’re using ClickBank and you read or hear the phrase “Postback URL” in ClickMagick documentation or videos, that is the same thing as ClickBank’s Instant Notification URL.
 
Caution: ClickBank’s Instant Notification feature is not compatible with ClickMagick’s advanced “Entry Link” setting. If you’re using the “Entry Link” setting, you’ll need to use ClickBank’s “Integrated Sales Reporting” method to track conversions. You can read more about that here:
