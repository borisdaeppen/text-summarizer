How do I geoblock a group of countries?

If you want to block clicks from certain countries from ever going to any of your tracking links or rotators, you can set up a system-wide geoblock within your User Profile.

Just go to the Account menu, select Profile, and from there, click on the Traffic tab. Scroll down and look for this setting:



Select the countries you want to block, and set a redirect URL that tells ClickMagick where you want to send those unwanted clicks to, and you’re all set.

No more unwanted clicks!


Using Geotargeting to Geoblock

If you don’t want set up a system-wide geoblock, but only want to geoblock on a particular tracking link or rotator, you can do that using the geotargeting settings in the advanced settings of any link or rotator. You can read more about that in our FAQ on geotargeting:
