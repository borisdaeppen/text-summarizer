What does “Skip URL Verification” do?

Whenever ClickMagick asks you for a URL, it automatically checks the URL to make sure that it points to a valid page and gives you a warning if page looks bad. This is what you want to happen 97% of the time so that you can correct the URL.

But there are situations where entering a currently bad URL is exactly what you want to do, perhaps because you’re getting ready to create the page that it points to, or you know that the website is currently down.

Whatever the reason, you can override ClickMagick’s URL verification check by clicking on “Skip URL Verification.”
