Can I track sales when promoting affiliate offers?

The only way you can properly track sales when promoting affiliate offers is if the affiliate system or platform supports one of the 3 industry-standard methods of conversion tracking …


Tracking Pixels – The easiest way to track conversions.

Some affiliate networks/offers will allow you to add your own tracking pixels, and this is by far the best way to go if you can. Many don’t allow this however.

If you can, you’ll want to read this FAQ:
 
  How do I track conversions using Tracking Pixels?


Postback URLs – Some of the larger affiliate networks support Postback URLs which is another way you can track your affiliate conversions.

To learn how to track sales using Postback URLs, please see this special tutorial:
 
   How do I track conversions using Postback URLs?


Sub-ID Tracking – The last option is Sub-ID tracking, which most decent affiliate programs support.

To learn how to track your affiliate sales using only Sub-IDs, please see this tutorial:
 
   How do I track affiliate conversions with Sub-IDs?


And finally, if whatever you’re promoting doesn’t support any of these three methods of conversion tracking, you’ll probably want to use what we call …
 
  The “Bonus Page” Workaround Method
