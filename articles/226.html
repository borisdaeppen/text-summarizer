Can I have multiple usernames for my account?

By design, ClickMagick allows only one username for each account.

You can, however, have multiple custom domains for each account which is probably what you’re really looking for anyway. It’s waaaay cooler.

If the term “custom domain” is new to you, it simply means that you can create a bunch of catchy website names like these:

www.myamazinggolfclubs.com
www.bestcarpolish.net
www.guiltyquilting.info
etc.

You can create an unlimited number of these website names (called “domains”) and easily direct them all to your single ClickMagick account.

We have a comprehensive FAQ and video walkthrough that will show you exactly how to set up your own custom domains. The only tricky part is coming up with names that haven’t been taken yet.

Happy hunting!
