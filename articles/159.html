FAQ Checker Test Cases—Should report errors—Don’t fix!

This FAQ has a number of common errors that we want the FAQ debugger to automatically find and report for us. Don’t remove these tests so that we can verify that we don’t break the debugger when we add more tests to it.

First, let’s test a correct FAQ link:
 
  How do I set up a custom domain using Script Forwarding?


Next, let’s test a proper LINK link which should open in a separate window or tab:
 
  Click here for Bing’s full list of parameters


Now let’s try a properly formatted video link:
 
How to use the Advanced Entry Link Setting
