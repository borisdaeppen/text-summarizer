How do I clone a funnel in the Funnel Builder?

Cloning a funnel in the Funnel Builder requires cloning both the funnel in ClickMagick and the actual funnel web pages themselves. This is a 4-step process and you must complete all 4 steps.

If you skip any of the 4 steps, you’ll either break the original funnel being cloned or the cloned funnel won’t be functional. Here’s why …

When you use the “Clone” command in the funnel’s Edit menu , only the funnel in the Funnel Builder is cloned, not the actual web pages that make up that funnel—the tracking code in those web pages still point to the original funnel you created in the Funnel Builder. 

To fully clone a funnel, you need to take these steps from the main Funnels page:
 
Step 1: Clone the funnel you created in ClickMagick using the “Clone” command. Simply click on the Edit menu to the right of the funnel you want to clone and choose the “Clone” command.

Step 2: Using your Page Builder, duplicate each of the web pages in your funnel. This is important because you will need to change the tracking code in the ((head)) section of these web pages.

Step 3: Copy the tracking code for the new funnel you created in Step 1 by choosing “Tracking Code” in the Settings menu to the right of the funnel’s name. You need to paste this tracking code into the ((head)) section of each of the new pages you created in Step 2, replacing the tracking code of the original funnel. This is necessary so that when visitors land on the new pages, the clicks will report back to the cloned funnel, not to the original one.

Step 4: You will need to create new Entry Links for the cloned funnel. Simply click on the Settings menu  again, this time choosing “Entry Links”. Create the new entry links just as you did for the original funnel.

 
Note: If you simply want to clone the funnel in the Funnel Builder in order to experiment with different visual layouts, that’s perfectly fine. You can clone the funnel as many times as you’d like and make changes to them as long as you realize that the “active” funnel is the one whose tracking code you’ve pasted into the ((head)) section of your web pages.
