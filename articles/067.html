What about creating Facebook custom audiences?

Easy. You’ve got two options …


Option 1: Create One Custom Audience

The first option is if you want to build only one custom audience to retarget a single group of people.

To do this, go to the Audiences section of the Facebook Ad Manager. From there, click on the “Create Audience” menu and choose “Custom Audience”. Next, choose “Website Traffic” then confirm that “All website vistors” is selected in the drop-down menu. Give the audience a name and click the “Create Audience” button.

That’s it. You’re done and you can start promoting your tracking links.


Option 2: Create Multiple Custom Audiences

The second option is if you want to create multiple custom audiences.

Before setting things up with Facebook, you’ll first need to get the “Link ID” number for each tracking link that you’ll be using to promote with Facebook. You can find the Link ID number in the lower-right corner of each tracking link when you have it open for editing:



 

Next, for each separate Audience that you want to set up, follow these steps:
 
1.    Go to the Audiences section of the Facebook Ad Manager and click on the “Create Audience” menu. Select “Custom Audience” then choose “Website Traffic”.
 
2.   From the dialog box, select “People who visited specific web pages” in the drop-down menu. You’ll be asked to enter the URL of the page associated with this Audience. Enter the URL for the tracking link you’re going to use for this audience. You need to use this URL with number replaced with one of the Link ID numbers you copied down earlier:

https://www.clkmg.com/redir.cgi?lid=number
 
 
Caution: This special URL is only intended to be used in rare situations …  It is NOT meant to be accessed or promoted directly.

Be sure you use the URL above exactly as shown—using www.clkmg.com—even if you’ve set up your own custom tracking domain!
 
 

Using the tracking link in the screenshot shown earlier, the number would be 590328 and your Facebook entry should look similar to this:


 
 
3.   Give your Audience a name and click the “Create Audience” button.
 
4.   Add your Facebook pixel to the Pixel/Code field in Advanced Settings of the tracking for this Audience.
 

You can repeat this process as many times as you need to to create multiple custom audiences in Facebook.

Once you’ve set up your Facebook custom audiences as described above, you simply promote your ClickMagick tracking links as usual.
