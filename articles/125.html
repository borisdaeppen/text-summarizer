How do I get started with Organic tracking?

To start tracking your Organic traffic all you need to do is add a bit of code to all the pages you want to track.

This video will walk you through the process …
 
Introduction to Organic Tracking


To get the tracking code for your project simply go to the Organic page, select the project you need the code for, then select “Get Tracking Code” from the Manage menu in the upper-right corner of the page.

Most website builders and content management systems make it easy to add tracking code to all your pages with one setting. If you’re not sure how to add code to your pages, read through this FAQ and it will show you many examples of how to do this:

Adding ClickMagick Code to Secure Pages

If you need to add the ClickMagick tracking code to secure pages, just change http to https in the URL of the tracking code.
