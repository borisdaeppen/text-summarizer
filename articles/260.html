How do we prioritize “bad clicks”?

When a user adds an IP address in their IP Manager, that “user” setting overrides any classification that the system has already attached to the IP.

So, for example, if the IP is currently flagged as an “Abuser” by the system and Abuser clicks are getting blocked, if the user then adds that IP to their IP Manager, that IP is now a “User” click and would be handled according to the setting for User clicks.

This means that if the User clicks are set to “Nothing” the clicks will be passed on to the Primary URL even though the system may have previously classified the IP as an Abuser.

It’s also important to remember than any clicks that are automatically flagged as an Abuser will only be flagged for 24 hours.
