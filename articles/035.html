How can I do “content locking” with pop ups?

“Content locking” is a tactic where you force the user to enter their email address or take some other action before they can access certain valuable content.

It’s a great way to build your list, get more Facebook “likes”, and things of that nature …

… and you can easily do this with ClickMagick pop ups which we call MagickPops.

Just set up a tracking link with the Primary URL pointing to a pre-sell or some other page, then cloak the link and add a pop up “on load” with an opt-in form for example.

And make sure that you set the pop up’s “Closable” setting to “No” for maximum effect otherwise the user will be able to easily bypass the pop up.

The pop-up will “dim” and obscure the page behind it, so the user has no choice but to interact with the pop up.

You would then set the “Thank You” page URL after they opt-in to the URL of the content the user is trying to access, and voilà—that’s content locking.
