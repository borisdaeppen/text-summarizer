How do I find the Funnel ID for my funnel?

Finding the Funnel ID for any funnel is really simple—you just go to the Tracking Code page for that funnel and you’ll find the ID listed there. Just follow these two steps:
 
1.   Go to the Funnels Page and click on the Settings menu  to the right of the funnel you’re interested in and select “Tracking Code”:
 



 
2.   The Tracking Code page will show you the Funnel ID right under the actual tracking code:


 
Caution: Do not copy the Funnel ID number shown in the screenshot above. You must use the number that is shown for your funnel in your account.


Every funnel has its own unique Funnel ID, so be sure to select the correct funnel to ensure that you’re getting the correct Funnel ID.
