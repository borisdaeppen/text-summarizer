How do I Troubleshoot a FourPercent Funnel?

This is a quick checklist of things to make sure that a custom FourPercent Funnel is set up correctly. When people complain that their funnel isn’t recording stats or actions aren’t showing up, one of these would be the reason:
 
1.    Does the Funnel Opt-in Page match the page the person actually lands on?

For the funnel to track a visitor, the person must come in through the Entry Link, and that entry link should be the same URL as the page they land on.

To get the URL of the Opt-in page, click on the funnel name on the Funnels page. This will open the Funnel Builder. Move your mouse over to the first page and choose Settings.

You’ll get the URL for the opt-in page.

Open that in a new tab and see if the URL changes to anything else. If it does, then the clicks to this funnel won’t be tracked.

This usually happens because ClickFunnels (where most FourPercent members build their custom pages) has a generic URL for the page and then the actual URL. What we need is the actual URL.

The FAQ below has details on how to set that up to match:
 
  Why does my funnel not show any visits?
 
 
2.    Is the person redirected to the correct Thank You URL after signing up?

FourPercent has a specific URL that it sends people to and redirects them to the Sales page. We need to make sure that the correct Funnel ID is passed through to the Sales page and that sales are recorded for the custom funnel.

First, opt-in on the user’s Opt-in page in Chrome. Using the Redirect Path plugin, copy the complete redirect path to the clipboard. Paste this into a text editor like Notepad on Windows or TextEdit on the Mac.

Next, use our URL tool to generate the correct Thank You page URL for their custom funnel that you’re checking:
 
  URL Tool For FourPercent


Doing a Find in your text file for that exact URL will show you if they’re redirecting to the correct link.

If they have a Bridge Page, the button on that bridge page should link to this URL.
 
 
3.    Do the Opt-in and Thank You pages have the correct Funnel Tracking Code?

When you clone a funnel, the new funnel will have a different Funnel ID than the original which means the Opt-in page and the Thank You page (if there is a Thank You page) will need to have that new code added.

To check that, click on the Settings menu for the funnel and choose Tracking Code.

Open the Opt-in and Thank You pages in a new tab. Right click anywhere and view the source code.

Searching for the funnel’s ID should help you find and compare the code.

If you want to be 100% sure, you can use this tool to compare the two pieces of code:
 
  Diff Checker
 
 
4.    Are you (or the customer) behind a proxy or VPN?

If you are, then your clicks would be flagged automatically and not show up in the UC count. Only legitimate clicks are counted as actual clicks. Disable your VPN and retest their entry link. That will usually start recording your clicks.
 
 
5.    Does the Entry Link they’re sending traffic to still exist?

This is rare, but sometimes users delete their entry links and still send traffic to them from some source (say, a solo ad they bought some time ago). You can check that by pasting the link, or just the cmfel value, into the CMFEL lookup tool on the Admin page.
 
 
6.    Are they looking in the right place?

Some people will create an entry link and then create a tracking link that redirects to that entry link. Then, they’ll expect to see actions on the Links page for the tracking link.

This won’t work, since it’s the Funnel Tracking Code that’s on their pages and not an Action Tracking Pixel. Ask them to check the Actions in the stats on the Funnels page (after confirming with them about where they are looking exactly).
 

You’re now a FourPercent Funnel Ninja.
