What’s the URL for the hidden Pixel/Code page?

When you use the Pixel/Code option of a tracking link to fire pixels or execute JavaScript code, ClickMagick creates a temporary, behind-the-scenes, web page that it uses to fire the pixels or execute the JavaScript code. After the pixels fire and the JavaScript code executes, this hidden page then redirects to the Primary URL (or split test URL) of the tracking link.

The hidden page has this URL ...

https://www.clkmg.com/redir.cgi?lid=number

… where number is the Link ID number that you can find in the lower-right corner of the tracking link when you have it open for editing:
