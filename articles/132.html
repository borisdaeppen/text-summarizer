What’s the risk of cloaking a rotator?

Some web pages do not allow themselves to be “cloaked” or “masked” so you have to be careful when cloaking a rotator.

The risk with cloaking a rotator, especially if you’re selling “solo ads,” is that it’s very easy to quickly add URLs to the rotator and forget to test each individual URL to make sure that it works when cloaked.

If you have a rotator URL that is showing a blank white page, that’s a dead giveaway that the URL cannot be cloaked. This is so common that we have a separate FAQ on the topic:
