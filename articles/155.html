How do I change the “speech bubble” in a MagickBar?

One of ClickMagick’s pre-designed MagickBar templates allows you to create a cool little “talking head” graphic that will float above the top or bottom of any web page.

For example, here’s what one of these MagickBars would look like in the MagickBar editor: 





That “speech bubble” is really just a simple background image so there aren’t any settings within ClickMagick to change its size.

If you want to change the size of the bubble, you simply need to replace the image in the MagickBar with a speech bubble that has a different size.

You can either copy and edit the default image, or you can search online for “speech bubble” or “thought bubble” and use an image that someone else has already created.
