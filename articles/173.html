Can I use ClickMagick to shorten links?

You sure can!

Link and URL shortening became popular when Twitter counted every character in your URL against your tweet length. That’s all changed, but link shortening can still be valuable for marketing reasons.

All you need to do is create your own “branded” custom domain, set it up with ClickMagick, and then you’ll be able to use your custom domain for as many short links as you want. You can be up and going in just a few minutes by following the instructions in this guide:
 
  How to Set up a Custom Domain with ClickMagick


Once you’ve set up your custom domain, you can create as many links as you want and they’ll all look something like this:

yourdomain.com/shortname

So, let’s suppose you grabbed the domain cool.io. Your links could then look like these:

cool.io/jKl8e
cool.io/golf
www.cool.io/xbox
etc.

If you’ve never created a ClickMagick link before, this video will show you how simple it is to do:
 
How to Create a Link with ClickMagick
 
 
Tip: Once you set up your links, not only will you have nice short links to use for marketing, but you’ll also be able to track all the statistics for your links.
