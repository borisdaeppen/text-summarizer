Why was my AdWords ad disapproved?

If your AdWords ad was disapproved, first check out their guidelines here:
 
  AdWords Policies for Acceptable Ads

If you review the AdWords Policies and you discover that your product or service does not meet their guidelines, then there’s not much you can do except to promote a different offer or use a different Ad Network that will allow you to promote your offer.

You should also note that AdWords does not allow cloaked links or cloaked domains. More on this in just a bit ...


Make Sure your Ad is Set Up Correctly

AdWords has strict policies on what your links must look like and there can’t be any misleading or misdirected website addresses or URLs. To configure your ad correctly, follow the instructions in this FAQ:
 
  How do I track my Pay-Per-Click Keywords?

 

Make Sure You’re using Uncloaked Links and Domains

Adwords does not allow links and domains to be cloaked and it will disapprove your ad if it detects that you are, in fact, cloaking.

To check if your link is cloaked, simply edit the link and make sure the Cloak Link option is set to No.

If you’ re using a custom domain, go to the Domain Manager and confirm that the domain you’re using with AdWords is set to Redirect:
 


 

If your domain is set to Cloaked, you may want to set it to Redirect, but it might be better to create a new subdomain that you use just for AdWords. For example, you might use adw. as the subdomain instead of www. or track. For example:

http://adw.yourdomain.com/linkID

After you add your new subdomain, remember to add adw.yourdomain.com to the ClickMagick Domain Manager as an uncloaked domain. If you forget this step, you will get an error when you try to use the domain.


Create an Uncloaked Custom Domain

If you’ve followed the previous steps and your ad was still not approved, it’s probably because you’re using your generic ClickMagick account URL …

http://www.clkmg.com/$username/linkID

Sometimes our generic tracking domain www.clkmg.com gets temporarily “blacklisted” by AdWords because someone tried to use it to promote something AdWords doesn’t like. Not only does their ad not get approved, but “their” domain (our www.clkmg.com domain) also gets put on the “naughty” list.

This doesn’t happen often, and we’re usually able to get it resolved relatively quickly, but at the end of the day, we can’t guarantee that our generic tracking domain will always work with AdWords …

Our advice here is to do what experienced online marketers always do ...

No matter what service they are using, experienced online marketers avoid using domains that are shared with other marketers—it’s just too easy for new online marketers to unintentionally get the shared domain blacklisted.

Instead, experienced online marketers always set up their own custom domains.

It’s quick and easy to set up a custom domain so there’s no reason not to do it. Just follow this guide and you’ll be set up in minutes:
 
  What is the recommended way to set up a custom domain?


Once you’ve set up a custom domain you won’t be affected by “problems” caused by other users, plus you’ll get all the other benefits of using a custom tracking domain. Just remember, the domain must be uncloaked if you’re going to use it with AdWords.
