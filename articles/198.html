How do I set up server-to-server tracking with CJ Affiliate?

To set up up server-to-server tracking between CJ Affiliate and ClickMagick, you’ll need to ask your CJ Affiliate account manager to install a ClickMagick Postback URL in your CJ Affiliate account. This is not something you can do yourself.

Other than that, you can learn everything else you need to know about setting up a Postback URL for server-to-server communication by reading this FAQ:
 
   How do I track conversions using Postback URLs?


If you already fully understand how Postback URLs work, this CJ Affiliate document might be useful to you as well:
 
  How to set up a Server-to-Server communication with CJ Affiliate


Again, just keep in the mind that you cannot set up the Postback URL in your CJ Affiliate account by yourself. Your account manager will need to do that for you.
