How can I control link cloaking with Script Forwarding?

When you generated your custom files for Script Forwarding, you selected whether you want your custom domain tracking or rotator links to be cloaked or not by default.

… but you can also override this default setting very easily on a per-link basis any time you want.

Consider the following custom domain tracking link:

http://$linkdomain/linkID

To force cloaking, regardless of the default setting you chose, simply add /cl to the end of the link, like this:

http://$linkdomain/linkID/cl

And to force NO cloaking, just add /nc like this:

http://$linkdomain/linkID/nc

It’s as simple as that!

If you’re using Sub-IDs with your links, just make sure that the /cl or /nc goes between the link name and the Sub-ID.

So if your Sub-ID was bingad, it should look like this:

http://$linkdomain/linkID/cl/bingad
