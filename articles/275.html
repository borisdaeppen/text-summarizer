What’s the best way to track Traffic Sources?

If you want to track your traffic sources, there are several effective ways to do it, depending upon your particular situation, and whether you’re using tracking links, or the FunnelMagick system:
 
1.    You can use a meaningful Sub-ID with your tracking link that shows where the click originated. This is a simple and easy approach unless you want to track the costs and ROI of your traffic sources. If you just want to track clicks, using a Sub-ID can be effective. You can read more about that here:
 
  Can I use Sub-IDs with my tracking links?
 
2.    You can also use multiple tracking links, one for each traffic source. If you want to track your costs and your ROI—and you’re not using the FunnelMagick system—you can set up a funnel using multiple tracking links. Setting this up can be a bit complicated so we have a complete walkthrough here:
 
  How do I track a funnel using Tracking Links?
 
3.    Finally, you can use the FunnelMagick system and create multiple entry links. This is by far the easiest and most flexible way to track multiple traffic sources within ClickMagick. You can learn how to set up multiple traffic sources with FunnelMagick by following this video:
 
How to Track a Funnel using FunnelMagick
