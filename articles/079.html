How can I report an IP address I think is a bot?

If you come across what you think is a bot that our system isn’t aware of yet, we’d really appreciate it if you could report it to us so we can take a look and add it to our auditing system if necessary.

To report an IP address you think is suspicious, simply click on the IP address and then click the “Report it” link right near the top of that page.

Once you report an IP address it will undergo a manual review and if we determine it’s a bot we’ll add it to the global system-wide click auditing system.

With your help our bot detection system will continue to get smarter and smarter as each day passes, and we thank you in advance for reporting any suspicious IPs.
