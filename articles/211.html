Why can’t I change my funnel’s Page URL setting?

If the Page URL setting for your Entry Page is disabled, that means that you have created entry links that still point to the old Page URL setting.

If you were to change the Page’s URL, you would break all of the existing entry links—at least from a funnel tracking viewpoint—because those entry links wouldn’t point to the new URL that you’re wanting to enter right now. Changing your page’s URL would also completely invalidate all of your current statistics.

This is why FunnelMagick won’t let you casually change your Page URL.

If you want to change the URL of your entry page, there are 3 approaches you can take, depending upon what you’re trying to accomplish. If you’re unsure which approach to use, use the first approach where you simply delete your Entry Links and start fresh.
 
 
1.    If you know that none of the current entry links are being used anywhere, you can simply go to the Settings menu  of your funnel and select “Entry Links”:




From there, you can delete your existing entry links. FunnelMagick will then let you edit the Page URL of the entry page for your funnel.
 
Caution: Keep in mind that if you’ve created any tracking links that point to your original entry links, you’ll need to update those tracking links. Once you correct your entry links, those tracking links will no longer be valid.

 
 
2.   If you have active entry links for the funnel, or if you’re unsure whether you do, another approach is to clone your existing funnel and make changes there.

Cloning a funnel is more complicated but is often the right approach. Cloning is more complicated because you’ll need to add the tracking code for the cloned funnel to the actual web pages of the funnel. Depending upon your situation, this might mean having to duplicate your web pages as well.

To make sure you don’t miss a step when cloning a funnel, we have a step-by-step guide here:

3.   Yet another solution is to create a “page variation” where your existing funnel has more than one entry page. With this setup, you keep the existing Page URL, and add another one that you can use to create new entry link URLs. Simply click on the “Variation URLs” tab in the Page Settings:
