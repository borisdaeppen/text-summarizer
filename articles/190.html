How do I customize the opt-in page for the Challenge Funnel?

To create and customize your own opt-in page for the FourPercent Challenge funnel, you’ll first need to install the standard Challenge funnel that Vick and his development team have created for you. If you haven’t taken this step already, just follow the steps in this FAQ:
 
   How do I create my FourPercent Challenge Funnel?
 
 
Note: To create a cloned funnel, you need to be on ClickMagick’s Standard Plan or Pro Plan. If you are on the Starter Plan, which is the least comprehensive plan, you cannot create a cloned funnel.

If you are on the Starter Plan and want to customize the opt-in page for the Challenge Funnel, you will need to upgrade your plan. Upgrading is easy to do—just go to the Billing Page and change your subscription:
 
  Go to My Billing Page



Create Your New Opt-In Page

Once you have cloned your Challenge funnel, you will need to use your Page Builder to create a new opt-in or landing page to replace the one in the standard funnel.

If you’re not already using a Page Builder, ClickFunnels is a very popular Page Builder that many people working with FourPercent use.

Once you have your FourPercent Challenge funnel installed and a replacement opt-in page designed, just follow the steps below to set up a custom opt-in page in your Challenge funnel, or watch this video which will walk you through the entire process, step-by-step:
 
How To Create a Custom Opt-In Page for Your Challenge Funnel


Written Walkthrough
 
1.    Clone Your Funnel. Your FourPercent Challenge funnel is read-only so that you can’t accidentally break it and lose sales. So, to install a customized opt-in page, you’ll first need to copy the standard Challenge funnel using the funnel Clone option. You can find the clone command in the Edit menu to the right of any funnel:



 
Caution: You may get a message that says you have already created the maximum number of funnels allowed in your plan. To upgrade your account to include more funnels, simply go to the the Account menu at the top of every page and choose the Billing option. You can upgrade your account from there.
 
 
 
2.   Change Your Opt-in Page in the Funnel Builder. Right now, the first page in your cloned funnel still points to the original opt-in page. To change that, go into the Funnel Builder and click on the Settings option for the first page in the funnel:



Simply change the page URL to point to your replacement opt-in page.

 
3.   Add the Funnel Tracking Code to the new Opt-in Page. Every page in your funnel needs to have the ClickMagick funnel tracking code installed on it in the ((head)) section of the page.

The Challenge funnel already has this tracking code installed on its pages, but your new opt-in page does not so you’ll need to add it to your page. To get the code, simply go to the Settings menu to the right of the new funnel you created in Step 1 above and select “Tracking Code”:



If you’re unfamiliar with how to add tracking code or JavaScript code to a web page, the following FAQ will show you how to do it for all of the popular Page Builders:
 
   How do I add tracking pixels or JavaScript code to [...]?
 
 
4.   Add the Thank You Page URL to your opt-in form. You’ll need to install an email capture form that you create in your GetResponse autoresponder account. The critical part here is that you will need a custom Thank You page link to use with your email optin form.

To create the special Thank You page URL for your FourPercent Challenge funnel, click on the button below and choose the funnel you cloned from the stardard Challenge funnel.


         


Copy the new Thank You page URL and add it to your email opt-in form. If you need help with this step, you will need to review the FAQs that your autoresponder service provides.
 
Note: If you also want to add an additional “bridge” page between your opt-in page and the FourPercent sales page, then set the Thank You Page URL of your autoresponder to point to your bridge page, and use the link created by the button in this step as the URL between the bridge page and the FourPercent Sale Page. In fact, you can insert as many pages as you want before the FourPercent Sales Page. as long as the final page uses the Thank You Page URL created in this step. Just remember to insert the funnel tracking code from Step 3 in all of your new pages.

 
5.   Create an Entry Link to the new funnel. Finally, your funnel is set up and you’re now ready to promote it. All you need is an Entry Link. If you don’t know how to create an Entry Link, you should review the video on Installing the standard Challenge Funnel. That video shows you exactly how to create Entry Links so you can promote your customized Challenge funnel.
 

That’s it! You should now have a fully functional customized Challenge Funnel.
