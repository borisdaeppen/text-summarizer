What is link “cloaking” and when do I need it?

Link cloaking is a way to “hide” the URL you’re promoting—for example an affiliate link.

If you don’t want people to know you’re sending them to an affiliate link, for example, just turn on cloaking and end users will only see our clkmg.com tracking domain (or they’ll see your own custom domain if you’ve added one and enabled it for cloaking).

You’ll also need to turn on cloaking if you want to add a MagickPop or MagickBar to a link, or if you want to supply “meta” tags like a title, description and image URL for sharing your tracking links on Facebook.

 
Note: In ClickMagick, we use the word “cloak” rather than “mask” in our documentation, so if you’re looking for topics on “masking,” search for the word “cloak” instead. You’ll get better results.
