Should I issue a refund for a customer?

This is a fairly common question from anyone just getting started with ClickMagick support so to answer this question, it's important to keep in mind that ClickMagick has never refused a refund to anyone for any reason...

With this in mind, that should eliminate any gray area for this question to conclude that you should refund a customer if they’re asking for a refund (even if it’s from 6+ months ago).
