How do I set up RTX with ClickMagick?

RTX supports Postback URL tracking, so you can send back conversion data to RTX.

Here’s how to set it up:
 
1.    Create a ClickMagick tracking link that you’ll use with RTX. Set the Primary URL of this link to the offer you’d like to promote.
 
 
2.    Set up your campaign and ad in RTX, and then add /{click_id} to the end of the tracking link you created in step 1:

http://www.clkmg.com/$username/targetlink/{click_id}

RTX will replace the {click_id} token with a unique value that ClickMagick will send back to RTX in the custom postback.
 
 
3.    Next, create the custom postback using the Custom Postbacks tool in the Tools menu. In the Which Link? drop-down, choose the tracking link that you created in step 1. Choose Sales as the Conversion Type, and set the Postback URL to your RTX S2S code, customized with ClickMagick’s [s1] and [amount] tokens. It should look something like this:

https://p.trkjmp.com/conv?v=OGU1YTM0M2MwZjA2ODM4ODYmE5NQ%3D%3D&c=[s1]&r=[amount]
 
Caution: Be sure to use the actual RTX S2S Postback URL for your campaign in your RTX account. You can read about the RTX requirements on the RTX website here:
 
  Track conversions with RTX using server-to-server tracking
 
 
4.    Click on Add Custom Postback.

Now, whenever anybody clicks on an RTX ad and a sale is made, ClickMagick will report the sale back to RTX.
