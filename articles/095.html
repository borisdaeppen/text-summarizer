What is the Domain Manager used for?

The Domain Manager can be found in the Tools menu at the top of any page, and it’s where you complete the final step of the custom domain set up process by adding your custom domain to the ClickMagick system.

There’s more you need to do though …

… so before using the Domain Manager make sure you following the instructions for setting up a custom domain using CNAME / A Records or Script Forwarding.
